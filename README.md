# 玩美學 Being Perfect 網站

玩美學 Being Perfect 是教育共生平台網站，採靜態 HTML / CSS / JavaScript 建置。

## 頁面

- `index.html`：首頁
- `about.html`：關於我們
- `courses.html`：課程介紹
- `talks.html`：共學講堂
- `workshops.html`：家長工作坊
- `partners.html`：教育夥伴
- `blog.html`：部落格
- `contact.html`：聯絡我們

## 本機預覽

可直接開啟 `index.html`，或使用本機伺服器：

```bash
python3 -m http.server 4173
```

然後開啟 `http://127.0.0.1:4173/index.html`。
